Asst. Front End Developer

No. of Vacancies: 02

Job Summary

Seeking two talented and motivated Assistant Front End Developers to join our dynamic team. The ideal candidates will be proficient in modern web technologies and eager to grow in a supportive environment.

Job Responsibilities

  • Develop and maintain responsive, user-friendly web applications using HTML, CSS (Bootstrap, Flexbox, Grid), JavaScript, jQuery.
  • Demonstrate a foundational understanding of Object-Oriented Programming (OOP), preferably with C#.
  • Possess familiarity with React (Redux, Next.js, etc.).
  • Understand AJAX and HTTP tools like XMLHttpRequest, fetch, or Axios for asynchronous operations.
  • Show eagerness to learn and contribute to .NET MVC and .NET Core Web API projects.
  • Basic knowledge of Canvas API, Chrome Extensions, or JavaScript Worker Services is advantageous.
  • Foundational understanding of Kubernetes and container orchestration.
  • Basic knowledge of monitoring tools such as Grafana and Prometheus.
  • Comprehend the functionality of Single Page Applications (SPA).
  • Participate in code reviews, maintain coding standards and best practices.
  • Understanding of Infrastructure as Code (IaC) using tools such as Terraform and Ansible.
  • Eager to learn and contribute to ASP.NET MVC, Blazor (optional), and enterprise-level web applications.
  • Be hardworking, responsible, and committed to meeting deadlines, taking ownership of assigned tasks and delivering high-quality results on time.

Educational Requirements

  • BSc in Computer Science & Engineering (CSE) or Software Engineering (SE)

Experience

  • 0 to 1 year of relevant experience

  • Age: Maximum 27 Years
